# Trabalho de AAIB
### Autores:
+ Bernardo Teixeira
+ Miguel Andrade
+ Sara Lobo

### Resumo:
Este trabalho consiste na implementação de um classificador que permita fornecer infomarção aos médicos se determinado paciente necessita de cadeira elétrica ou automática, com base nos valores de aceleração. De forma a enrriquecer o nosso trabalho identificaram-se sete labels, incluindo as anteriores, que são as seguintes:
1. Parado
2. Teste Inválido
